Приложение для визуализации алгоритмов поиска

Описание:

Проект демонстрирует реализацию и анализ нескольких алгоритмов поиска путей в графах, включая:

A* (поиск с эвристикой),
Алгоритм Беллмана-Форда,
Дейкстра,
Флойда-Уоршелла.

Проект акцентируется на сравнении производительности и эффективности использования памяти этих алгоритмов, что позволяет исследовать их практическое применение в задачах поиска оптимальных путей.

Навыки:

  • Разработка алгоритмов на Python.
  • Использование библиотек для профилирования памяти и времени выполнения (memory-profiler, psutil).
  • Создание визуализаций с PyQt6.
  • Работа с файлами и структурами данных (матрицы смежности, графы).

CRUD приложение на Go

Описание:

eToDo — это минималистичное приложение для управления задачами (todo-list), которое полностью разрабатывается на Go для серверной части и использует HTML + CSS + JavaScript для фронтенда.

Цель — создать легковесное, высокопроизводительное и надежное приложение, доступное как для локального использования, так и для развертывания в веб-среде.

Функциональность (планируется и реализуется):

  • Серверная часть на Go.
  • Хранение задач в базе данных PostgreSQL).
  • RESTful API для взаимодействия фронтенда и бэкенда.
  • Авторизация и управление пользователями (регистрация, вход).
  • Фронтенд на HTML + CSS + JavaScript.
  • Визуализация списка задач с функциями добавления, редактирования и удаления.
  • Использование Docker для контейнеризации и развёртывания фронтенда и бэкенда приложения.
